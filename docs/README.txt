Compilation:
Build with the provided gradle files and run in your android emulator of choice.


Currently implemented features:

Start page for user to self-identify if they're a caretaker or a patient.
Caretaker registration that allows signup to firebase.
Caretaker sign in to an already made account to firebase.
Setting page which allows the caretaker to switch to caretaker mode with password
In the caretaker mode, user can upload pictures, edit pictures, add new family member
upcoming: all family member displayed in list view with avatar
